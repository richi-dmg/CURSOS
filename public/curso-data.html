Descripción Del Curso

Este curso práctico te enseñará a trabajar con datos y automatizar tareas repetitivas utilizando Python y sus poderosas bibliotecas. Aprenderás a extraer, transformar, analizar y visualizar datos de múltiples fuentes, desde archivos Excel y CSV hasta bases de datos y APIs web. Dominarás técnicas de web scraping, crearás scripts de automatización que te ahorrarán horas de trabajo manual, y desarrollarás dashboards interactivos para presentar insights de manera efectiva. Cubre herramientas esenciales como Pandas para manipulación de datos, Matplotlib y Seaborn para visualización, y Selenium para automatización de navegadores.

Duración

8 semanas · 40 horas

Qué Vas A Aprender?

  • Análisis de datos con Pandas y NumPy
  • Visualización de datos con Matplotlib y Seaborn
  • Web scraping con BeautifulSoup y Selenium
  • SQL para análisis y consulta de bases de datos
