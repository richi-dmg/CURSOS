Descripción Del Curso

Este curso te introducirá al fascinante mundo del desarrollo web desde cero, sin necesidad de conocimientos previos. Comenzarás aprendiendo HTML5 para estructurar contenido web de manera semántica, luego dominarás CSS para dar vida y estilo a tus páginas, incluyendo técnicas modernas como Flexbox y Grid para crear layouts profesionales y responsive. Posteriormente, te adentrarás en JavaScript para añadir interactividad y dinamismo a tus sitios web.

Duración

8 semanas · 40 horas

Qué Vas A Aprender?

  • Crear páginas web con HTML5 semántico
  • Diseñar layouts modernos con CSS, Flexbox y Grid
  • Agregar interactividad básica con JavaScript
  • Publicar y deployar tus proyectos en línea

Módulos

  • Módulo 1: Introducción a la Web
  • Módulo 2: HTML5 – Estructura de páginas
  • Módulo 3: CSS3 – Estilos y diseño visual
  • Módulo 4: Diseño Responsive
  • Módulo 5: JavaScript básico
  • Módulo 6: Formularios e interacción
  • Módulo 7: Publicación de sitios web
  • Módulo 8: Proyecto final
