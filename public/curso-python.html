Descripción Del Curso

Este curso está diseñado para llevarte de cero a programador Python de forma práctica y efectiva. Python es uno de los lenguajes de programación más populares y versátiles del mundo, utilizado en desarrollo web, ciencia de datos, inteligencia artificial y automatización. Comenzarás con los fundamentos del lenguaje: sintaxis, variables, tipos de datos y estructuras de control, avanzando gradualmente hacia conceptos más complejos como funciones, módulos y programación orientada a objetos.

Duración

7 semanas · 35 horas

Que Vas A Aprender

  • Sintaxis fundamental de Python y estructuras de datos
  • Funciones, módulos y paquetes
  • Programación orientada a objetos en Python
  • Bibliotecas esenciales: NumPy y Pandas básico

  • Módulo 7: Automatización básica
